## 📌 Project Overview
This project is a **production-ready personal portfolio website** designed and deployed using modern **DevOps practices**.

Beyond a simple website, this project demonstrates:
- Continuous Integration & Continuous Deployment (CI/CD)
- Cloud-based hosting and delivery
